### 1. 用户认证模块

#### 1.1 启动页

#### 1.3 注册

### 2. 账号管理模块

#### 2.1 查看账号信息
- [ ] 显示用户名（不可修改）
- [ ] 显示昵称、年龄、性别、个性签名
- [ ] 显示当前头像

#### 2.2 修改昵称
- [ ] 点击"昵称"弹出编辑框
- [ ] 输入新昵称可以保存
- [ ] 修改后立即生效
- [ ] 超过20字提示错误

#### 2.3 修改年龄
- [ ] 点击"年龄"弹出编辑框
- [ ] 输入数字可以保存
- [ ] 非数字输入不保存
- [ ] 修改后立即生效

#### 2.4 修改性别
- [ ] 点击"性别"弹出选择框
- [ ] 可以选择"男"或"女"
- [ ] 修改后立即生效

#### 2.5 修改个性签名
- [ ] 点击"个性签名"弹出编辑框
- [ ] 可以输入多行文本
- [ ] 显示字符计数（0/140）
- [ ] 超过140字不能继续输入
- [ ] 修改后立即生效

#### 2.6 修改密码
- [ ] 点击"修改密码"弹出编辑框
- [ ] 需要输入当前密码
- [ ] 需要输入新密码两次
- [ ] 当前密码错误提示错误
- [ ] 两次新密码不一致提示错误
- [ ] 修改成功后需要重新登录

#### 2.7 更换头像
- [ ] 点击头像区域弹出选择框
- [ ] 可以选择"拍照"
- [ ] 可以选择"从相册选择"
- [ ] 可以选择"预设头像"
- [ ] 预设头像有12个选项
- [ ] 修改后立即生效
- [ ] 头像显示正常（预设图标或自定义图片）

#### 2.8 清除数据
- [ ] 点击"清除数据"弹出确认对话框
- [ ] 确认后删除所有交易记录
- [ ] 交易记录列表变空

#### 2.9 注销账号
- [ ] 点击"注销账号"弹出确认对话框
- [ ] 确认后删除账号
- [ ] 自动退出登录
- [ ] 跳转到登录页

### 3. 记账功能模块

#### 3.1 添加交易（记一笔）
- [ ] 点击底部"+"按钮可以添加交易
- [ ] 可以选择"收入"或"支出"
- [ ] 可以选择金额（支持小数）
- [ ] 可以选择分类
- [ ] 可以选择日期（默认今天）
- [ ] 可以输入备注（可选）
- [ ] 点击"保存"添加成功
- [ ] 保存后返回列表页
- [ ] 空金额不能保存

#### 3.2 编辑交易
- [ ] 在交易详情页点击"编辑"
- [ ] 可以修改金额、分类、日期、备注
- [ ] 可以修改收入/支出类型
- [ ] 保存后更新成功
- [ ] 列表中数据同步更新

#### 3.3 删除交易
- [ ] 在交易详情页点击"删除"
- [ ] 弹出确认对话框
- [ ] 确认后删除成功
- [ ] 返回列表页
- [ ] 列表中数据已删除
