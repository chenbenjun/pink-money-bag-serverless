# 手机访问说明

## 访问地址

请在手机浏览器中访问以下地址：

**前端应用**：
```
http://9.129.115.216:5000
```

**交易详情测试页面**：
```
http://9.129.115.216:5000/test-transaction-detail.html
```

## 前提条件

1. **手机和电脑在同一局域网内**
   - 连接同一个 Wi-Fi
   - 或通过有线连接到同一个路由器

2. **电脑防火墙配置**
   - 确保 5000 和 9091 端口允许外部访问
   - Windows：Windows Defender 防火墙 -> 允许应用通过防火墙
   - Mac：系统偏好设置 -> 安全性与隐私 -> 防火墙

3. **网络连接正常**
   - 电脑 IP：9.129.115.216
   - 端口：5000（前端）、9091（后端）

## 常见问题

### Q: 手机无法访问
**A**:
1. 检查防火墙设置
2. 确认 IP 地址正确
3. 确认手机和电脑在同一网络

### Q: 前端能访问但 API 调用失败
**A**:
1. 检查后端服务是否运行：`curl http://9.129.115.216:9091/api/v1/health`
2. 检查 CORS 配置（已配置）
3. 查看浏览器控制台错误信息

### Q: IP 地址变化了
**A**:
1. 重新获取 IP：`hostname -I | awk '{print $1}'`
2. 更新 `.env` 文件中的 IP 地址
3. 重启前端服务
